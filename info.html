TO DO LIST - Informazioni

Come funziona la To Do List?

Questa applicazione è progettata per aiutarti a organizzare e gestire le tue attività in modo semplice e veloce.

Guida alle Icone

  • Icone di Stato delle Task
  • Pallino Rosso: task non completata (attiva)
  • Pallino Verde: task completata
  • Icone di Navigazione
  • Casa: torna alla pagina principale
  • Task Completate: visualizza le task finite
  • Icone di Filtro
  • Persona: filtra per utente
  • Cartella: filtra per categoria
  • Icone di Azione sulle Task
  • Più: aggiungi nuova task o sottotask
  • Lista: visualizza sottotask
  • Post-it: visualizza dettagli della task
  • Matita: modifica task o sottotask
  • Cestino: elimina singola task o sottotask
  • X Ottagono: elimina tutte le task completate
Conclusione: La To Do List è pensata per essere veloce, intuitiva e personalizzabile. Che tu stia gestendo progetti, studio o impegni personali, è lo strumento ideale per non dimenticare nulla.
